# adventure time transcripts

These are all the transcripts scraped from the [wiki](https://adventuretime.fandom.com).

## Gap Filling

Some episodes were missing or incomplete on the wiki. These gaps were filled by converting SRT subtitle files to wiki format. See [TRANSCRIPT-GAPS.md](TRANSCRIPT-GAPS.md) for details on which episodes were converted and corrections made.

## Verification

SRT conversions lack speaker labels and require manual verification. See [VERIFICATION-WORKFLOW.md](VERIFICATION-WORKFLOW.md) for the process used to verify speaker attributions.
